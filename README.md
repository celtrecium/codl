# codl
Cursor-Optimized Drawing Library.

This is curses-like library, written on C. At the moment there is only support for GNU/Linux.

# Build and install

GNU/Linux:

1. `mkdir build/`
2. `cd build`
3. `cmake ..`
4. `make`
5. `sudo make install`

# Uninstall
Run `sudo xargs rm < install_manifest.txt` command
